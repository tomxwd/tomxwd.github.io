03_Gradle_gradle的安装

gradle官网

gradle下载地址

下载bin.zip

下载好后解压，进行环境变量配置：

变量 值
GRADLE_HOME D:\environment\gradle-5.6;
Path %GRADLE_HOME%\bin;

在cmd中输入gradle -v，出现版本信息则安装、配置成功。
