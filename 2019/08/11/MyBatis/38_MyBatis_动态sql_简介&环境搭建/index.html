38_MyBatis_动态sql_简介&环境搭建

官网—动态sql

简单说就是不同的条件变化不同的sql语句；

简介

  • 动态SQL是MyBatis强大的特性之一，可以极大简化我们拼装sql的操作；
  • 动态SQL元素和使用JSTL或其他类似基于XML的文本处理器相似；
  • MyBatis采用功能强大的基于OGNL表达式来简化操作
    • if
    • choose(when,otherwise)
    • trim(where,set)
    • foreach
